## Description

A simple Web App to view Weather Data of different cities. Built to learn how RESTful API works.

It uses the [** OpenWeatherMap API **](https://openweathermap.org/api) for weather information.

## Tech Stack

- Django
- Bulma CSS
- OpenWeatherMap API
