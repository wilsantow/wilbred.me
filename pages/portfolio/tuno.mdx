## Description

The Application enables users to view their top tracks and artists in different time ranges.
It uses the [** Spotify Web API **](https://developer.spotify.com/documentation/web-api/) for accessing user data.

It follows the ** Implicit Grant ** among the four Authorization flows mentioned in the Spotify Documentation for user Authorization.

## Tech Stack

- ReactJS
- Tailwind CSS
- Framer Motion
- Spotify Web API
